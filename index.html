function cargarDatos(mesIndex,year){ query("INGRESOS",ingresosData=>{ query("EGRESOS",egresosData=>{ let ingresos=ingresosData?calcularPorMes(ingresosData,mesIndex,year):0; let egresos=egresosData?calcularPorMes(egresosData,mesIndex,year):0; let diferencia=ingresos-egresos; // Formato con separadores de miles document.getElementById("ingresos").innerText = "C$ " + ingresos.toLocaleString('en-US',{ minimumFractionDigits:2, maximumFractionDigits:2 }); document.getElementById("egresos").innerText = "C$ " + egresos.toLocaleString('en-US',{ minimumFractionDigits:2, maximumFractionDigits:2 }); document.getElementById("diferencia").innerText = "C$ " + diferencia.toLocaleString('en-US',{ minimumFractionDigits:2, maximumFractionDigits:2 }); // Crear tabla para gráfico const chartData = new google.visualization.DataTable(); chartData.addColumn('string','Tipo'); chartData.addColumn('number','Monto'); chartData.addRows([ ['Ingresos', ingresos], ['Egresos', egresos] ]); // Formato monetario en gráfico const formatter = new google.visualization.NumberFormat({ prefix: 'C$ ', groupingSymbol: ',', fractionDigits: 2 }); formatter.format(chartData,1); const chart = new google.visualization.ColumnChart( document.getElementById('chart_div') ); chart.draw(chartData,{ title:`Reporte Financiero - ${meses[mesIndex]} ${year}`, legend:{position:"none"}, vAxis:{ format:'#,###.00' }, tooltip:{ textStyle:{ fontSize:12 } } }); }); }); }
